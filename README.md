# Keypoint Counting Classifiers (KCC)

Codebase for **Keypoint Counting Classifiers** — a few-shot image classification method that counts how many learned visual keypoints appear in a query image.

**Paper:** Wickstrøm, K., et al. *Keypoint Counting Classifiers*. arXiv:2512.17891, 2024.
https://arxiv.org/abs/2512.17891

## Getting Started

The introductory notebook loads example bird images from Wikimedia Commons and sets up DINOv2 feature extraction as the first step of the KCC pipeline.
